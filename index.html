Phase - a melodic loop machine. Phase is a fun experimental composition tool that plays sequences of 12 notes in direct counterpoint to the same sequence in different phase positions. Features:

- responsive design
- simple visual interface for composing short melodies
- vast number of permutations available by sequencing 12 note melodies.

My Tracks

RTSEQS128 - a 3 minute track produced in Italy. Most of RTSEQS128 was coded in PureData and SonicPi (SuperCollider) and then mixed with Ardour. The piece features a set of chords that have a fluctuating pulse, in a kind of wave pattern, and layered over this there is a melody and a more percussive track that slides between perceptible pulse and a tone.
